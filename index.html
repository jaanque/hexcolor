Adivina el Color HEX

Las reglas son simples: adivina el código de color HEX oculto en 6 intentos. Cada intento debe ser un código HEX válido de 6 caracteres (0-9, A-F).

Después de cada intento el color de las casillas cambia para mostrar qué tan cerca estás de acertar el código:

A
VERDE significa que el carácter está en el código y en la posición CORRECTA.
B
AMARILLO significa que el carácter está presente en el código pero en la posición INCORRECTA.
C
GRIS significa que el carácter NO está presente en el código.

El color que se muestra arriba corresponde al código que debes adivinar. Intenta identificar el código basándote en tu percepción del color y los resultados de tus intentos.
